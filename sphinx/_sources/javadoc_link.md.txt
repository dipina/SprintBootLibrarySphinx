# Javadoc API Documentation

Full Javadoc is generated by the **maven-javadoc-plugin** during `mvn site` and published as part of the GitHub Pages deployment.

## Accessing the Javadoc

```{important}
**[→ Open Javadoc](../site/apidocs/index.html)**

*(Link works when browsing the published GitHub Pages site)*
```

## Key Documented Classes

The following classes have rich Javadoc comments that are worth exploring:

| Class | Package | Description |
|---|---|---|
| `BorrowingController` | `controller` | REST endpoints for borrowing operations — includes `@Operation` (OpenAPI) and Doxygen-style doc |
| `BookService` | `service` | Business logic for book inventory management |
| `Borrowing` | `model` | JPA entity — models a borrowing transaction |
| `BookRepository` | `repository` | Spring Data JPA repository with custom query methods |
| `UserService` | `service` | User management operations |

## Generating Javadoc Locally

Javadoc is generated as part of the Maven site:

```bash
mvn site
# Open: target/site/apidocs/index.html
```

Or standalone:

```bash
mvn javadoc:javadoc
# Open: target/reports/apidocs/index.html
```

## Relationship with Doxygen

The project uses **both** Javadoc and Doxygen:

| Tool | Output | Strength |
|---|---|---|
| **Javadoc** | `site/apidocs/` | Standard Java API reference; IDE integration |
| **Doxygen** | `doxygen/html/` | Cross-referenced call graphs, class diagrams (with Graphviz), richer navigation |

Both are published to GitHub Pages and linked from this Sphinx portal.
