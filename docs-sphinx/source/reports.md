# Test & Quality Reports

All reports below are generated automatically by the CI/CD pipeline and published alongside this Sphinx portal on GitHub Pages.

## JaCoCo — Code Coverage

JaCoCo instruments the bytecode and measures which lines, branches, and methods are exercised by the test suite.

```{note}
**Direct link →** [JaCoCo Coverage Report](../site/jacoco/index.html)
```

The project enforces a minimum **25 % line coverage** via `mvn verify`. Raising this threshold is recommended as test suites grow.

## Surefire — Unit Test Report

The Surefire plugin records every test case result (pass / fail / skipped) and execution time.

```{note}
**Direct link →** [Surefire Unit Test Report](../site/surefire-report.html)
```

## Checkstyle — Code Style

Validates source code against Sun/Google coding conventions. Violations are listed with file and line number.

```{note}
**Direct link →** [Checkstyle Report](../site/checkstyle.html)
```

## PMD — Static Analysis

Detects common coding problems: unused variables, empty catch blocks, overly complex methods.

```{note}
**Direct link →** [PMD Report](../site/pmd.html)  
**Direct link →** [CPD (Copy-Paste Detector)](../site/cpd.html)
```

## JDepend — Package Metrics

Measures coupling, cohesion, and instability at the package level. Useful for spotting architectural drift.

```{note}
**Direct link →** [JDepend Report](../site/jdepend-report.html)
```

## Performance Test Report

Generated only when running the `performance` Maven profile. Shows throughput and 99th-percentile latency for annotated test scenarios.

```{note}
**Direct link →** [Performance Report](../site/reports/perf-report.html)
```

## Javadoc API Reference

Full Javadoc for all public classes and methods, generated by `maven-javadoc-plugin` as part of `mvn site`.

```{note}
**Direct link →** [Javadoc](../site/apidocs/index.html)
```

## Doxygen Technical Reference

Doxygen parses the Java source and produces a cross-referenced HTML reference. Several classes include rich Doxygen-formatted comments:

- `BorrowingController.java`
- `BookService.java`
- `Borrowing.java`

```{note}
**Direct link →** [Doxygen HTML Reference](../doxygen/html/index.html)
```

---

## Generating All Reports Locally

```bash
# 1. Unit tests + JaCoCo
mvn test jacoco:report

# 2. Performance tests
mvn -Pperformance integration-test

# 3. Copy perf report into site directory
mvn -Pperformance resources:copy-resources@copy-perf-report

# 4. Generate the full Maven site (Javadoc, Checkstyle, PMD, JDepend, Surefire)
mvn site

# 5. Copy everything to docs/ and build Sphinx
mvn post-site
cd docs-sphinx && make html
cp -r _build/html/* ../docs/sphinx/
```
